This project is to allow people to use ArgoUML in Chinese.

Jeff Liu is controlling the project.

Work plan

  • Translate the ArgoUML GUI (zh_TW). Jeff Liu is working with that.
  • Get the translation included into the ArgoUML distribution. Linus Tolke is working with that.

Mission

The goals of this project is to:

  • Create a localization of ArgoUML into Traditional Chinese zh_TW.
  • Become the center of the Chinese ArgoUML community.
  • Create a localization of ArgoUML into Simplified Chinese zh_CN.
